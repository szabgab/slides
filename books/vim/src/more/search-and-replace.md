# Search and replace: substitute


* /
* s

```
:%s/Old/New/              Replace Old by New.
:%s/^    /\t/             Replace leading 4-spaces by a tab.
:%s/$/./                  Put a . at the end of every line.
:'&lt;,'&gt;s/...         Subtitute in range.
```
