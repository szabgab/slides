# Switching buffers

* :ls
* :bn
* :bp
* :bN
* :bd
* :b TAB


[Working with buffers](http://vimcasts.org/episodes/working-with-buffers/)

```
:ls   list buffers

:bn     next buffer
:bp     previous buffer

:bN     switch to buffer N
:b TAB  cycle thorugh the names of the files in the buffers

:bd     delete current buffer
:bd3    delete buffer number 3
```
