# Get request headers using curl


Using [curl](https://curl.se/)

```
$ curl https://httpbin.org/headers
```

I got the following response:

```
{
  "headers": {
    "Accept": "*/*",
    "Host": "httpbin.org",
    "User-Agent": "curl/7.81.0",
    "X-Amzn-Trace-Id": "Root=1-6394b68a-65bd2c145c2304274a314120"
  }
}
```
