# Exercise: sort mixed string

In a file we have the string where each string has
a single letter at the beginning and then a number.
Sort them based on the number only, disregarding the letter.

Input:


```
A4
B1
A27
A3
```

Expected output:


```
B1
A3
A4
A27
```
