# if-statement

* if
* else



If there is only one statement inside the block of the if-statement, then there is no need
to put curly braces. Just like in other C-like languages.


```
if (cond)
   statement

if (cond) {
   statement;
   ...
   statement;
}

if (cond) {
   ...
} else {
   ...
}
```
