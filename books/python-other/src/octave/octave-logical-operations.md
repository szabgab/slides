# Logical operations

```
0        % false
1        % true

==       % Is equal?
~=       % Is NOT equal?
3 == 3   % 1
3 == 4   % 0
3 ~= 3   % 0
3 ~= 4   % 1
1 && 0   % 0  (logical and)
1 && 2   % 1  (logical and)
1 || 0   % 1  (logical or)
~1       % 0  (logical not)
~2       % 0
~0       % 1
```
