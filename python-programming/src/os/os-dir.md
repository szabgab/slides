# OS dir (mkdir, makedirs, remove, rmdir)


* mkdir
* makedirs
* remove
* unlink
* rmdir
* removedirs
* rmtree
* shutil

* `mkdir` is like `mkdir` in Linux and Windows
* `makedirs` is like `mkdir -p` in Linux
* `remove` and `unlink` are like `rm -f` in Linux or `del` in Windows
* `rmdir` is like `rmdir`
