# Locate the Child Windows


```perl
printf "Main      %6s %6s %6s %6s\n", GetWindowPos($Main);

my @children = GetChildWindows $Main;

foreach my $id (@children) {
  printf "%6s  %6s %6s %6s %6s\n", $id, GetWindowPos($id);
}
```
